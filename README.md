# Best-Practice-WPF

Windows Presentation Foundation (*WPF*) *是*一个可创建桌面客户端应用程序的UI 框架。

WPF使用xaml布局页面，Windows最新框架WIN UI3 或 MAUI都采用xaml布局技术，以上三种框架很容易过渡。
